> **CattleIQ** is a full-stack AI-powered dairy farm management platform. It combines a **Linear Regression milk predictor** and an **XGBoost disease detector** inside a clean Flask web app — giving farmers actionable health and yield intelligence through a bilingual, mobile-friendly interface.

## 🧠 Machine Learning Architecture

---

### 🥛 Model 1 — Milk Yield Prediction

```
INPUT FEATURES
──────────────────────────────────────────────────────────────
  Feed_kg          →  Daily feed quantity       [5  – 25  kg]
  Temp_C           →  Ambient temperature       [15 – 40  °C]
  Humidity         →  Relative humidity         [30 – 90   %]
  Milking_Time_min →  Duration of milking       [5  – 30 min]
  temp_humidity_ratio → Engineered feature (Temp × Humidity)

PIPELINE
──────────────────────────────────────────────────────────────
  Raw inputs
      │
      ▼
  Outlier clipping (IQR-based)
      │
      ▼
  StandardScaler normalization
      │
      ▼
  Linear Regression (with feature engineering)
      │
      ▼
  OUTPUT: Predicted Daily Milk Yield (Liters)

PERFORMANCE
──────────────────────────────────────────────────────────────
  MSE               :  0.0317
  R² Score          :  0.9940   ← 99.4% accuracy
  Cross-val R²      :  0.9942
  MAPE              :  0.71%
```

---

### 🏥 Model 2 — Disease Detection (XGBoost)

```
INPUT FEATURES  (19 engineered health indicators)
──────────────────────────────────────────────────────────────
  Cow_ID                   │  Body_Temp
  Day_of_Month             │  Breed
  Months_After_Birth       │  + 13 sensor-derived features
  Previous_Mastitis_History│

PIPELINE
──────────────────────────────────────────────────────────────
  Raw health inputs
      │
      ▼
  Feature engineering (19 indicators)
      │
      ▼
  GroupKFold Cross-Validation  ← prevents data leakage across cows
      │
      ▼
  XGBoost Classifier (hyperparameter-tuned)
      │
      ├── Risk Score (probability)
      ├── Disease Classification
      └── Treatment Recommendations

PERFORMANCE
──────────────────────────────────────────────────────────────
  ROC AUC           :  99.29%
  Validation        :  5-Fold GroupKFold
  Features          :  19 engineered
  Tuning            :  Grid search hyperparameter optimization
```

> **Why GroupKFold?** — Standard KFold would leak data across records from the same cow. GroupKFold ensures no cow appears in both train and validation sets, giving honest generalization scores.

## 🌐 API Endpoints

| Method | Endpoint | Description |
|---|---|---|
| `GET` | `/` | Redirect to dashboard |
| `GET` | `/login` | Login page |
| `POST` | `/login` | Authenticate user |
| `GET` | `/dashboard` | Main farm dashboard |
| `GET` | `/milk_prediction` | Milk prediction form |
| `POST` | `/predict_milk` | Run LinearRegression inference |
